CUSTOM TEMPLATE INSTRUCTIONS
============================

To use a custom background for generated hiring images:

1. Create your background image:
   - Recommended size: 1080x1080 pixels (square, Instagram-friendly)
   - Use dark colors for good text contrast
   - Leave space in center for text overlay

2. Save it as:
   templates/template.png

3. The Content Factory will automatically use it instead of
   generating a gradient background.

Example template specs:
- Format: PNG or JPEG
- Size: 1080x1080 px
- Color: Dark blue (#1a365d) or similar
- Leave center 60% clear for text

The tool will add:
- "WE ARE HIRING IN [CITY]" text
- Job count
- Department names
- "APPLY NOW" call to action
